# homebrew-jot

Homebrew tap for [Jot](https://jot.ideaflow.page/) — free, open-source, on-device dictation for macOS.

## Install

```
brew install --cask vineetu/jot/mac
```

Requires Apple Silicon and macOS Sonoma 14.0+. The app auto-updates via Sparkle after install.

Source: https://github.com/vineetu/JOT-Transcribe
